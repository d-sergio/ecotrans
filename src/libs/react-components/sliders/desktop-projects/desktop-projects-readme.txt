Все изменения в slider.js помечены комментарием
/**********МОДИФИКАЦИЯ**********/

Используется create-always-active.js вместо стандартного create-slides.js, который
передаёт в центральный (относительно viewport) слайд пропс active={true}.
По этому пропсу элемент узнаёт, что должен изменить своё состояние

Добавлена функция updateArrowsCoords(), устанавливающая координаты кнопок
управления слайдером. Две кнопки (назад и вперёд) позиционируются относительно
горизонтального центра слайдера и смещаются соответственно влево и вправо от него
на расстояние, равное половине ширины центрального слайда + значение из
props.buttonShift. Проще говоря, это делается, чтобы расположить кнопки слева
и справа от центрального слайда.

[слайд1] <- [слайд2] -> [слайд3]
